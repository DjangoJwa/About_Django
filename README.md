Django
==

기본 세팅 (Windows 기준)
--


- 가상환경 세팅
```
C:/Python/tmp> python -m venv (your_environment_name)
```

- 가상환경 실행
```
C:/Python/tmp> (your_environment_name)/Scripts/activate
```

- Django Install
```
(venv) C:/Python/tmp> pip install django
```

- Project Start
```
(venv) C:/Python/tmp> django-admin startproject (your_project_name)
```
